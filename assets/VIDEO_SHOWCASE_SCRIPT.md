# Potato Shield – Full Application Video Showcase Script

Use this script to record a walkthrough video covering every part of the app. Each section has **screen**, **action**, **script/audio**, and optional **duration**. Speak naturally; timings are approximate.

---

## PRE-ROLL (Optional)
**Screen:** Black or logo  
**Script:** *“This is Potato Shield — AI-powered disease prediction and diagnosis for potato farmers in India and the UK.”*  
**Duration:** ~10 sec

## TIPS FOR RECORDING

- **One take per section:** Record each section separately, then cut together, so you can redo only the part that failed.
- **Pause before clicks:** Pause 1–2 seconds before each click so the viewer can see where you’re clicking.
- **Zoom:** Use browser zoom (e.g. 90% or 100%) so text and buttons are readable.
- **Resolution:** Record at 1080p if possible.
- **Cursor:** Keep cursor visible for clicks; optional highlight or circle for important buttons.
- **Backend:** Ensure the API (e.g. `localhost:8000`) is running so chat, dashboard, and diagnostic all return real data.

Total script length is roughly **6–8 minutes** at a natural speaking pace; you can shorten by dropping optional sections (Learn More, Register, Setup Field, or Diagnostic) for a 3–4 minute version.
